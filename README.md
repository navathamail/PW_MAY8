# PW_May8

## Project Description

This repository contains JavaScript practice files for the PW_May8 coding course. The project is organized by day folders (day1, day2, ...), each containing small exercises and examples covering fundamentals like functions, arrays, objects, loops, and asynchronous patterns.

## Prerequisites

- Node.js (optional, for running any Node-based examples)
- A modern web browser for client-side examples
- Git installed and configured

## Setup Instructions

1. Clone the repository:

```bash
# using HTTPS
git clone https://github.com/navathamail/PW_MAY8.git
```

2. Change into the project directory:

```bash
cd PW_MAY8
```

## Installation

No installation is required for the plain JavaScript files. If you plan to run Node-based examples, install dependencies (if any) using:

```bash
npm install
```

## Usage

- Open the HTML files in a browser to run client-side examples (e.g., `day1/index.html`).
- Run JS files with Node where appropriate:

```bash
node day12/practice.js
```

## Project Structure

- Each `dayN/` folder contains exercises for that day.
- Filenames indicate the topic (e.g., `01_namedfunctions.js`, `01_arraymethods.js`).

## Notes & Features

- This is primarily an educational collection of short example scripts to practice JavaScript fundamentals.
- Feel free to reorganize, add examples, or include tooling (linters, bundlers) as needed.
